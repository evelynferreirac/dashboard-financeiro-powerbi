# 📊 Dashboard Financeiro com Power BI

## 📌 Sobre o projeto
Este projeto consiste em um dashboard interativo desenvolvido no Power BI para análise de dados financeiros, com foco em monitoramento de receitas, custos, lucro e desempenho geral.

## 🎯 Objetivo
Fornecer uma visão clara e estratégica dos indicadores financeiros, permitindo identificar oportunidades de crescimento, controle de gastos e apoio na tomada de decisão baseada em dados.

## 🛠️ Tecnologias e ferramentas
- Power BI  
- Excel  
- Modelagem de Dados  
- ETL (Extração, Transformação e Carga de dados)  
- DAX (criação de medidas e cálculos)

## 📊 Indicadores analisados (KPIs)
- Recebimentos  
- Custos  
- Impostos  
- Lucro  
- Margem (%)  
- Volume de movimentações  

## 📈 Análises realizadas
- Evolução do lucro ao longo dos meses  
- Comparativo entre receitas e custos  
- Análise de margem de lucro  
- Distribuição de movimentações financeiras  
- Participação por instituições bancárias  

## 🧠 Insights gerados
- Mais de **44% das transações foram realizadas via PIX**, indicando forte adesão ao meio digital  
- Crescimento consistente do lucro ao longo dos meses  
- Identificação de períodos com redução de lucro  
- Melhor visualização da relação entre custos e rentabilidade
